fastent

custom models for named-entity recognition
pip install fastent

fastent is designed for end-to-end creation of named-entity recognition models.

The fastent Python library is built to scale to very large datasets.

To train a model for a new type of entity, you just need a list of examples.

You are not limited to predefined types like person, location and organization.

fastent includes integrations with libraries like spaCy and datasets like Reddit.
